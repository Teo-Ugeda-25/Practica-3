# Practica 3
## Comentarios en PHP
Los comentarios de PHP se ponen de esta manera.
````php
// Este es un comentario de una linea

/* 
Este es un comentario de varias lineas. Es ideal para explicar bloques logicos más complejos. 
*/
````

## Comentarios en JAVA
```java
/** 
* Clase de ejemplo para mostrar comentarios 
* @author Teo Ugeda
*/

public class EjemploComentarios {

    // a. Comentario de una línea

    /* b. Comentario de 
          varias líneas */

    public void miMetodo() {
        System.out.println("JavaDoc permite documentar autoría y métodos.");
    }

```
## Comentarios en C#
```C#

using System;

// Comentario de una linea 

public class Programa {
    /* Comentario de varias lineas. Ideal para documentar logica completa o bloques de codigo antiguos */
    public static void Main(){
        Console.WriteLine("Esto es C#");
    }
}

```
## Comentarios en JavaScript
```JavaScript
// a. Comentario de una línea para una variable
const mensaje = "Hola Mundo";

/* b. Comentario de varias líneas
   Se puede usar para describir el propósito
   de una función específica. */
function mostrarMensaje() {
    console.log(mensaje);
}
````
